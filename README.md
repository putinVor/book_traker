# 📚 Book Tracker - Трекер прочитанных книг

**Автор:** Каримов Марсель

## Описание программы

Book Tracker — это графическое приложение для ведения списка прочитанных книг. Программа позволяет добавлять книги, фильтровать их по жанру и количеству страниц, а также сохранять данные в JSON-файл.

## Функции

- ➕ Добавление книг (название, автор, жанр, количество страниц)
- 🔍 Фильтрация по жанру (точное совпадение)
- 🔍 Фильтрация по количеству страниц (больше указанного значения)
- 🗑️ Удаление книг из списка
- 💾 Автоматическое сохранение в `books.json`
- 📊 Удобное табличное отображение

## Как запустить

### Требования:
- Python 3.6 или выше
- Tkinter (входит в стандартную установку Python)

### Запуск:
```bash
python book_tracker.py
